更新日志

工具新增功能

  1. 新增 重力感应调试支持。
  2. 新增 地理位置调试支持。
  3. 新增 <video/> live 属性，支持调试 src 为 m3u8 直播视频资源。

工具bugfix

  1. 优化 上传代码速度慢的问题。
  2. 修复 wx.switchTab 不触发 onShow 的问题。
  3. 修复 wx.getUserInfo 用户取消授权时返回的 errMsg 与客户端不一致的问题。
  4. 修复 wx.downloadFile 的 url 中带有 query 时，返回的 localId 无法使用的问题。
  5. 修复 wx.previewImage 不支持 localId 的问题。
  6. 修复 wx.stopBackgroundAudio 后没有回调 `wx.onBackgroundAudio
  7. 修复 navigationBarTextStyle 配置在工具上不生效的问题。
  8. 修复 在 tabbar 页面调用 wx.navigateBack 时异常的问题。
  9. 修复 编辑模块在目录上右键出现菜单不一致的问题。
  10. 修复 拼音输入法在编辑时中文显示不全的问题。
  11. 修复 用户取消授权后还能继续弹窗的问题。
  12. 修复 在 windows 系统任务栏右键关闭时会 crash 的问题。
  13. 修复 ES6 编译报错没有显示的问题。
  14. 修复 工具异常退出以后可能会导致无法启动的问题。
